# mkcareer-notes

本仓库用于归档和发布多个独立的 HTML 页面。页面内容主要围绕职业发展、技能笔记与个人学习记录，不针对任何特定平台或具体网站。

## 项目简介

mkcareer-notes 是一个静态 HTML 页面集合仓库。每个页面均为独立文件，可直接在浏览器中打开查看。项目本身不依赖任何后端服务，也不涉及动态数据交互，适合作为资料存档或个人笔记展示。

## 目录说明

```
.
├── index.html          # 仓库首页 / 入口页面
├── notes/              # 笔记类页面归档
├── projects/           # 项目记录页面归档
├── resources/          # 资源收集与整理页面
└── assets/             # 公共资源文件（样式、脚本等）
```

- `notes/`：存放各类学习笔记、职业思考等独立页面。
- `projects/`：记录个人项目经历、技术实践相关页面。
- `resources/`：整理收集的外部资料、工具推荐等页面。
- `assets/`：供各页面引用的公共 CSS、JavaScript 或图片文件。

## 页面归档说明

- 每个页面均为完整 HTML 文档，可独立运行。
- 页面之间无强依赖关系，可直接通过文件路径或本地服务器访问。
- 页面内容可能随时更新或补充，不保证所有页面始终处于最终版本。
